## Projects

**[Plot](https://github.com/dfelton42/group03-senior-project)** - iOS App, Group Senior Project (Fall 2025)
Built a live campus events feed where students post and interact with events in real time, developed with a team using Swift and Supabase. Owned the Google Maps integration and AI chatbot feature, the app also includes RSVPs, upvote/downvote functionality, asynchronous live updates, and multiple filterable views built collaboratively with the team.

**[Schedulr](https://github.com/ChrisChatel/CMSI-4072_AlfordChatelFelton)** - Meeting Scheduling Website, Senior Project (Spring 2026)
Built a meeting scheduling application that helps large groups find times that maximize attendance. Built with JS, HTML and Supabase. Rather than just displaying availability overlaps like tools such as When2Meet, the app analyzes availability data and ranks potential meeting times using algorithmic/AI-assisted techniques to surface clear, data-driven recommendations.

Designed an interactive narrative system focused on player choice and consequences, with branching paths and a moral score variable. Implemented NPC dialogue UI, triggers, camera transitions, and inventory logic in Lua. Organized modular script structure within Roblox Studio.
